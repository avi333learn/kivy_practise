# BOUNCING BALL GUI
---
### Concept:
When we press "GO" button, green ball starts moving. When ball touches edges of wall, it bounces in such way that it always remain visible on screen. <br/>
### Requirements:
1. python 3.6 to 3.9
2. kivy 2.0.0 <br/><br/>
